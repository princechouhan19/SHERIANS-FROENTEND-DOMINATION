CSS UNITS GUIDE

PX (Pixels)

Pixels are fixed-size units that provide precise control over element dimensions. They remain constant regardless of screen size or user preferences, making them ideal for elements that need exact sizing.

200px × 200px

% (Percentage)

Percentage units are relative to the parent element's dimensions. They're essential for creating flexible layouts that adapt to their container's size.

80% width

VW & VH

Viewport Width (vw) and Viewport Height (vh) units allow precise control over element sizing relative to the browser window. 1vw = 1% of viewport width, 1vh = 1% of viewport height.

25vw × 25vh

EM

EM units are relative to the current element's font size. They cascade through nested elements, making them perfect for scalable typography and spacing that maintains proportional relationships.

15em × 15em

REM

REM (Root EM) units are relative to the root element's font size, providing consistent sizing throughout the document. They're ideal for maintaining uniform spacing and typography across complex layouts.

15rem × 15rem

MAX

Max-width and max-height properties prevent elements from exceeding specified dimensions. They're crucial for responsive design, ensuring content remains readable and layouts don't break on larger screens.

max-width: 300px

MIN

Min-width and min-height properties ensure elements maintain minimum dimensions. They prevent content from becoming too small to read or interact with on smaller screens.

min-width: 250px
